
# Expense Tracker

Simple Flask app to track your reckless spendings on gacha games

## Requirements
```plaintext
flask
sqlite3
```

## Usage
First time users will need to initialize the database by running:
```python
python __init__.py
```
After that run:
```python
python app.py
```
